# KAKI LINK

KAKI LINK は、IoT・AI・コミュニティ機能を使って、柿の樹を複数人で見守り・育てることを想定したスマート柿園Webアプリのデモです。

## (企画書)[https://docs.google.com/document/d/10D8Qo2LepE-rljoa-gQeZ6fdqvN2UnLrI5z-N2ffLec/edit?usp=sharing
]

## 主な機能

- ホーム画面
- マイ柿の樹の状態確認
- 水やり操作デモ
- 自動水やり予約
- ライブカメラ映像デモ
- AI柿ちゃんとのチャット
- 共同管理者チャット
- 作業予約
- 成長記録
- ポイント特典交換
